Introducción¶

El descenso de gradiente es un algoritmo de optimizacin empleado para encontrar el mnimo de una funcin. En el mbito del Machine Learning, en concreto, se utiliza para encontrar los valores de los parmetros (pesos) de una funcin coste que minimicen el coste. Para ello se ir minimizando la funcin coste movindose iterativamente en la direccin del descenso ms pronunciado segn lo definido por la pendiente negativa del gradiente.

Regresión logística¶

En primer lugar, es importante comentar que la regresin logsitca es uno de los algoritmos de aprendizaje automtico ms simples y comnmente utilizados para resolver problemas de clasificacin binaria pues esta describe y estima la relacin entre una variable binaria dependiente y las variables independientes. Para proceder con este mtodo lo primero que debemos hacer es aprender a implementar la funcin sigmoidea.

La funcin sigmoidea o logstica definida como (1/(1 + e^-x)) es una funcin que da una curva en forma de 'S' que puede tomar cualquier nmero de valor real y asignarlo a un valor entre 0 y 1. Si la curva va a infinito positivo, y predicho se convertir en 1, y si la curva va a infinito negativo, y predicho se convertir en 0. Si la salida de la funcin sigmoidea es mayor que 0.5, podemos clasificar el resultado como 1 o s, y si es menor que 0.5, podemos clasificarlo como 0 o no.

In [6]:
# definimos la funcin
def sigmoid(z):
    """
    return the sigmoid of z
    """
    return 1 / (1 + np.exp(-z))

Función de coste¶

Mediante el mtodo de descenso de gradiente vamos a minimizar la funcin que mide el error de prediccin del modelo en el conjunto de datos, dicha funcin de error se conoce como funcin de coste. Gracias a esta funcin podemos saber cmo de bueno es nuestro modelo haciendo predicciones para un conjunto dado de parmetros. La funcin de coste tiene como objetivo encontrar el peso de los valores que minimizan el coste, y esto se consigue a travs de derivadas parciales.

La funcin costo siempre toma tres parmetros: theta, X e y donde theta es la posicin incial, X el vector de filas e y un vector categrico. Y su objetivo es encontrar la w y la b que minimizan el coste a travs de derivadas parciales.

In [8]:
# se define la funcion costo junto con sus parametros
def costFunction(theta, X, y):    
    """
    Takes in numpy array theta, X and y and return the logistic regression cost function and gradient
    """
    
    # longitud del vector categorico
    m = len(y)  
    
    # defino predicciones con la funcion sigmoidea y el producto escalar de las matrices X y theta
    predictions = sigmoid(np.dot(X,theta))  
    # defino error como resultado del logaritmo de la prediccion
    error = (-y * np.log(predictions)) - ((1-y)*np.log(1-predictions)) 
    cost = 1/m * sum(error)
    
    # donde X transpose es la matriz transpuesta del vector de filas 
    grad = 1/m * np.dot(X.transpose(),(predictions - y))  
    
    return cost[0] , grad

Normalización de las variables¶

Algunos algoritmos de aprendizaje automtico, como en este caso la regresin logstica, que utilizan el descenso de gradientes como tcnica de optimizacin requieren que los datos se escalen. La variable X al estar dentro de la frmula del descenso de gradiente afectar al tamao del paso del descenso. Para garantizar que este se mueva suavemente hacia los mnimos y que los pasos se actualicen al mismo ritmo para todas las variables, es necesario escalar los datos antes de introducirlos en el modelo.

Vamos a realizar el escalado mediante la normalizacin de la media, esta distribucin tomar valores entre -1 y 1 con una media igual a 0.

In [9]:
def featureNormalization(X):  
    """
    Take in numpy array of X values and return normalize X values,
    the mean and standard deviation of each feature
    """
    # media de la matrix X
    mean = np.mean(X,axis=0) 
    # desviacion estandar de la matriz X
    std = np.std(X,axis=0)    
    
    # matriz X normalizada 
    X_norm = (X - mean) / std  
    
    return X_norm , mean , std

Función descenso de gradiente¶

El descenso de gradiente es un algoritmo que permite localizar un mnimo en una funcin de forma iterativa, en Machine Learning dicha funcin a minimizar es la funcin coste (definida anteriormente) que reduce la diferencia entre el modelo que buscamos y la salida que tiene que aprender. Las iteraciones se producen entre un valor actual y una posicin, y cada nueva posicin que obtenemos al dar pasos poco a poco se calcula con la posicin actual menos un paso hacia la direccin de mayor descenso en esa posicin.

La funcin del descenso de gradiente tiene las entradas X (vector de filas), y (vector categrico), theta (poscin), learning rate (tamao de los pasos), nmero de iteraciones (nmero de pasos).

In [11]:
# definimos la funcion de gradiente para que minimize la funcion coste mediante derivada
def gradientDescent(X, y, theta, learning_rate, num_iters):
    """
    Take in numpy array X, y and theta and update theta by taking num_iters gradient steps
    with learning rate of alpha
    
    return theta and the list of the cost of theta during each iteration
    """
    
    m=len(y)
    J_history =[]
    
    # bucle que vaya calculando para cada iteracion el coste y el gradiente en cada posicion 
    for i in range(num_iters):
        cost, grad = costFunction(theta,X,y)
        theta = theta - (learning_rate * grad)
        J_history.append(cost)
    
    return theta , J_history
